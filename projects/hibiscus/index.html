Repo: Hibiscus

Hibiscus Repositories

How to set up

  • Go to the Menu File -> Settings.
  • Chose the Tab Plugins.
  • Add the following site.
https://bmhm.github.io/repositories/hibiscus

Plugins

  • hibiscus.ffb.depot (Github)
    Plugin, which uses the ffb.depot.client library to implement FFB Balance. See below for library details.

Libraries

  • ffb.depot.client (Github)
    Jax-RS 2.0 Library, which can authenticate a user and parses data from the fidelity.de page to access ones funds depot.
  • slf4j-jameica (Github)
    slf4j-bridge. If installed, java libraries which depend on slf4j will log to jameicas logging facility (org.willuhn.logging.Logger).
